Survival
========

AS3 Zombie survival game made form many tutorials of the net, is in noway coded "good" :P

Move: W,A,S,D

Shoot: left click

Change weapons: 1,2,3,4

Pause Game: P


Current Weapons:
pistol
uzi
shotgun
flamethrower

Current Items:
medpack
speedpack
ammopack
